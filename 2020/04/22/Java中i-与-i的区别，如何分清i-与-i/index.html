使用自增操作符和自减操作符可以使表达式更加简短，但也会使它们比较复杂且难以读懂。比如上述代码中出现的 j = i++ + ++i + ++i + i++; 只是为了测试大家是否更好的掌握了i ++与 ++ i，但在实际编程过程中应该避免。

我的总结如下：

i++、i– ：若参与调用，输出值不 +1，否则 +1
++i、–i ：无论是否参与调用，输出值均 +1
