2.方法内存

方法代码片段属于.class字节码文件的一部分，在调用方法的时候才会在栈中分配内存，称为压栈；方法执行结束释放方法的内存空间称为弹栈。方法执行完之后，方法内局部变量的空间也释放，因此局部变量只在方法内有效。JVM的栈中主要存储的是局部变量。
